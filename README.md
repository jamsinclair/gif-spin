# gif-spin

[![Demo of GIF Spin app](demo.gif)](https://gifspin.netlify.app)

> Generate spinning gif animations from a static image in your browser!

## Development
1. Clone the repo
1. Run `yarn` to install dependencies
1. Run `yarn dev` to start development server
1. Make your changes?

## Future plans
